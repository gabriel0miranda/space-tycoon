# Endless eXpanse

Este é um jogo de exploração espacial bem simples e bastante ambicioso. O nome reflete suas principais inspirações: Endless Sky, pelo [pessoal do Endless Sky](https://github.com/endless-sky/endless-sky/graphs/contributors) The Expanse, por [James Sociedade Anônima Corey](https://www.jamessacorey.com/) e X, pela [Egosoft](https://www.egosoft.com/).

É feito com [Love2D](https://www.love2d.org/) em [Lua](https://www.lua.org/) e até agora tem o seguinte escopo/conteúdo:
