# Endless eXpanse

Este é um jogo de exploração espacial bem simples e bastante ambicioso. O nome reflete suas principais inspirações: Endless Sky, pelo [pessoal do Endless Sky](https://github.com/endless-sky/endless-sky/graphs/contributors) The Expanse, por [James Sociedade Anônima Corey](https://www.jamessacorey.com/) e X, pela [Egosoft](https://www.egosoft.com/).

É feito com [Love2D](https://www.love2d.org/) em [Lua](https://www.lua.org/) e até agora tem o seguinte escopo/conteúdo:

- Menu principal
    - Botões:
        [x] New game
        [ ] Continue
        [ ] Controls
        [ ] Options
        [x] Quit
    - Funcionalidade
        [x] Controle pelo teclado via números
        [x] Controle pelo teclado via direcionais
        [x] Controle pelo mouse
        [ ] Controle pelo gamepad
        [x] Inicia novo jogo
        [ ] Seleciona jogo salvo e continua
        [ ] Mostra controles disponíveis
        [ ] Permite edição de controles
        [ ] Mostra opções disponíveis
        [ ] Permite edição de opções
        [x] Sai do jogo
- Pilotagem
    - Funcionalidade
        [x] Controle pelo teclado
        [ ] Controle pelo mouse
        [ ] Controle pelo gamepad
        [x] Movimento de naves
        [x] Sistema de controle de reação (RCS)
- Sistemas Estelares
    - Entidades
        [x] Estrelas
        [x] Planetas
        [x] Estações espaciais
        [x] Asteróides
        [x] Buracos de minhoca
        [x] NPC
    - Funcionalidades
        [x] Movimento de asteróides (gravitação)
